1. Overview

CleanSwitch provides an automated, reliable method for cleaning VLANs, trunk ports, and configuration artifacts across dozens of switches without any manual SSH interaction.

The script is:

fully automated

self-contained (no external files)

safe (dummy data only in repo)

auditable through generated logs

The tool is suitable for repetitive cleanup in controlled network environments.
